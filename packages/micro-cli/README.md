## Usage


- Chalk：正确设定终端的字符样式
- Figlet：使用普通字符制作大字母的程序（LCTT 译注：使用标准字符，拼凑出图片）
- Inquirer：通用交互式命令行用户界面的集合
- ShellJS：Node.js 版本的可移植 Unix Shell 命令行工具
- meow：一个简单的命令行助手工具
- yargs：一个命令行参数解析工具
- pkg：将你的 Node.js 程序包装在一个可执行文件中。
